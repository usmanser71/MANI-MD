MANI-MD

*MANI-MD* is a powerful, multi-device compatible WhatsApp bot built using the Baileys library.

🚀 Features

- Full WhatsApp MD Support
- Auto-reply & commands
- Group moderation tools
- Media & sticker support
- Easy configuration

🛠 Setup Instructions

1. *Clone the repo:*
   ```bash
   git clone https://github.com/usmanser71/MANI-MD.git
   cd MANI-MD
   ```

2. *Install dependencies:*
   ```bash
   npm install
   ```

3. *Run the bot:*
   ```bash
   node index.js
   ```

4. *Scan QR Code with your WhatsApp*

📁 Folder Structure

- `commands/` → Bot features
- `lib/` → Core handlers
- `media/` → Audio/images
- `auth/` → WhatsApp session
- `index.js` → Main entry file
